# Naming Conventions
This is a quick overview of the approriate use cases of the different naming conventions

# Java
## Class Names
Should be in Capital Camel Case
- `MyClass.java`
- `YourClass.java`

## Variable Names
Should be in Camel Case
- `myVariable`
- `thisIsAVariable`

# Github
## Branches
Should be in Kebab case
- `my-branch-name`
- `this-is-a-branch`
